# 레시피 저장형 서비스 기능 명세서

버전: v1.1  
작성일: 2026-03-06  

## 1. 사용자 기능

### 1.1 인증
- 회원가입
- 로그인
- 비밀번호 재설정
- 로그아웃

### 1.2 레시피 기본 기능
- 레시피 생성
  - 제목, 설명, 재료, 조리 단계, 준비시간, 조리시간, 분량, 카테고리, 공개 여부 입력
- 레시피 조회
  - 내 레시피 목록 조회
  - 레시피 상세 조회
- 레시피 수정
  - 본인 소유 레시피만 수정
  - 부분 수정 지원
- 레시피 삭제
  - 삭제 전 확인 다이얼로그

### 1.3 공개/공유
- 레시피 공개/비공개 전환
- 공개 레시피 상세 페이지 공유 링크
- 비회원 공개 레시피 목록 조회(최신 공개순 20개 기본)
- 비회원 공개 검색(제목/재료 텍스트 기반, 공개 데이터만)

### 1.4 검색·정렬·필터
- 검색: 제목/재료 기준 텍스트 검색
- 정렬: 최신 등록순/최근 수정순/이름순
- 필터: 공개 여부, 카테고리

### 1.5 이미지
- 대표 이미지 업로드/미리보기
- 업로드 실패 시 재시도
- 이미지 교체/삭제

## 2. 운영자 관점
- Auth/DB 정책 상태 점검
- DB 사용량/요청량 이상징후 점검
- 정책 위반 데이터 검토(필요 시 비공개/삭제)

## 3. 우선순위
- Must
  - 인증, 레시피 CRUD, RLS 보안, 공개 목록/상세 규칙 적용
- Should
  - 검색/정렬, 이미지 업로드, 공유 링크
- Could
  - 태그 분리, 즐겨찾기, 레시피 버전 기록

## 4. 라우팅 가이드(권장)
- `/` : 공개 레시피 피드(최신 공개순)
- `/login`, `/signup`, `/reset-password`
- `/recipes` : 내 레시피 대시보드
- `/recipes/new` : 레시피 작성
- `/recipes/[id]` : 상세(소유자이면 편집 버튼 노출)
- `/recipes/[id]/edit` : 수정

## 5. 구현 방식(확정)
- App Router + Route Handler API (`/api/*`) 를 기본 경로로 사용
- 인증 유틸은 서버/클라이언트 모두에서 공유
